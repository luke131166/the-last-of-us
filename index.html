Project Idea

Overview

We would all love to work with our friends and build a community together, right? That is exactly what I plan to do. I love cars as much as other people love animals, to me they are something to care for, love and appreciate.

I would love to create a community/business that aims to help endorse charities and the general community through a multitude of business applications and media outlets.

The main branches of my project would include an array of different business prospects that use various forms of IT to create a wide business and community.

Motivation

My motivation to create a project like this is to integrate the many useful forms of IT to create a community and platform for people who love the same passion as me. There are many different places for people to find information, products and learn on many different websites, institutions and shops – however there is not a place that easily integrates all these aspects together to create a one-stop place for enthusiasts.

Description/Tools & Technologies

The general idea is to have the following all merged into a single platform –

• Online forum
• Website to purchase products (such as merchandise or parts).
• Online media/video outlet that helps to inspire, teach and entertain customers.
• A workshop/mechanic to fix, restore and/or modify cars.
• Interactive site which people can record, edit and view their own personal goals/achievements and share with others.

There is not a single place that people can go to find all these different products/outlets. Although there are many places’ people can find what they are looking for in the world of cars, there is no place in which people can go to successfully achieve all their goals and be able to share and collaborate with one another in all these fields.

The main idea is to have a website, which integrates forums, shops, social media, personal blogs, and contact points for people to physically purchase services from a qualified workshop.

The product would contain different IT applications in all these branches such as –

• A dedicated server is to be created for the website to be hosted on. This would increase the websites performance and be able to optimally handle the high load of the website through the amount of people that could be using it at the same time.
• An implemented phone/chat support that can assist people in need with exploring our many products and resources we have to offer.
• Have an easy-to-use interface which will successfully direct people to where they need to go – this will be achieved with a great Front End Developer, who’s job it is to make sure people are able to access the site efficiently and effectively.
• High server security using a Secure File Transfer Protocol (SFTP). With all the information that will be stored on the website, it is a must that this information is not stolen to be used by a third party – everyone’s privacy should remain safe.
• Daily backups of the server’s content, as to reverse possible changes that are made which can delete/destroy content that people upload, share or store (such as blogs).
• Integrated blog & forum using a third-party provider such as WordPress using a cPanel hosted server.
• The use of car tuning software in automotive engineering/development (prevalent at physical workshop). There are many great tuning platforms and software that can be used to do so, such as Toad Pro and Elite Software Programmer (ESP).
• Online shop in the website that integrates the use of payment providers such as PayPal, Visa & Mastercard. This way, it is accessible to most people to be able to buy any of the products, as PayPal can use most payment methods through their own services, which is then translated to the website as a secured method of payment. A similar software to Shopify can be used to achieve this, or a custom-made software that can create invoices, receive payments through the any of the provided methods and confirm orders for pick-up or delivery. Shopify is a great way to implement an online store into a website, with over 1,700,000 businesses worldwide that use the current product.
• Integrated media support can be used on the website through other media outlets such as YouTube, or create a platform that saves users uploaded data to the server to be played to other users on the website. A quick way to do this is to implement WordPress on the website to be able to integrate embedded links from external sources to be played on the page.

Skills Required

There are many skills required to bring this project to life. Webhosting is something that is most needed, followed by software such as WordPress or similar created software to implement videos, forums and blogs into the website, the usage of cPanel and SQL databases to store information and integrate third party sources into the website.

There needs to be a Front End & Back End developer who needs to have a variety of skills such as programming, ability to adapt code in different languages to create a smooth experience. There are many people with the skills required to build websites and use these

different platforms all around the world – integrating them into one is the main objective of this project.

There are lots of special hardware involved in this project, such as the physical hardware in the workshop, however lots of computers, network storage and server hardware are required to create a website that can function using all of these tools.

Outcome

If my project is a success, it will help to position car enthusiasts all around the world to be able to do what they love, explore, collaborate and connect with one another through a single platform where they can find the answers they are looking for or just enjoy community-based entertainment through videos, forums or blogs.

It will be able to be accessed from people all around the world, however the more physical workings of the business would for customers in Australia, as they would need to physically transport their cars to the service, however the online store would allow us to ship parts (if legal) to other countries all around the world with simple payment methods and delivery services being used.

It will impact the world as we know it, as it will create lots of opportunity for people around the world to be able to source information and products in one place, saving time and recourses for all involved.
